📊 Financial Loan Insights Dashboard


🔹 Overview
This project focuses on analyzing bank loan data to identify trends, evaluate risk, and understand customer behavior. An interactive Excel dashboard is created to provide clear insights into loan performance and support data-driven decision-making.

🔹 Objectives
Analyze loan approval and rejection patterns
Identify high-risk customer segments
Track loan performance and default trends
Provide actionable insights using data visualization

🔹 Tools Used
Microsoft Excel
Pivot Tables
Charts & Graphs
Data Cleaning & Transformation
etc

🔹 Key Insights
Identified factors influencing loan approval
Detected high default risk segments
Analyzed customer income and loan distribution
Observed trends in loan status over time

🔹 Features
Interactive dashboard with filters
KPI metrics for loan performance
Visual representation of key trends

🔹 Conclusion
The project demonstrates how Excel can be used effectively for financial data analysis and risk assessment, helping stakeholders make informed lending decisions.
